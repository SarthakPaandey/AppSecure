# CWE-209: Generation of Error Message Containing Sensitive Information

**Official URL:** https://cwe.mitre.org/data/definitions/209.html

## Description

The software generates an error message that includes sensitive information about its environment, users, or associated data — such as stack traces, SQL errors, or internal paths.

## Mitigations

- Return generic error messages to clients.
- Log detailed diagnostics server-side only with access controls.
- Disable verbose debug modes in production.
- Sanitize exception messages before any client exposure.
