# CWE-639: Authorization Bypass Through User-Controlled Key (IDOR / BOLA)

**Official URL:** https://cwe.mitre.org/data/definitions/639.html

## Description

The system's authorization functionality does not prevent one user from gaining access to another user's data or resources by modifying a key value such as an ID in the request.

## Mitigations

- Enforce server-side ownership/tenancy checks on every object access.
- Prefer indirect object references mapped server-side to authorized resources.
- Deny by default; never rely on unguessable IDs alone for authorization.
- Test horizontal and vertical privilege escalation systematically.
