# CWE-918: Server-Side Request Forgery (SSRF)

**Official URL:** https://cwe.mitre.org/data/definitions/918.html

## Description

The web server receives a URL or similar request from an upstream component and retrieves the contents of this URL, but it does not sufficiently ensure that the request is being sent to the expected destination.

## Mitigations

- Allowlist permitted hosts/schemes for outbound fetches.
- Block private, link-local, and metadata IP ranges (e.g., 169.254.169.254).
- Disable or carefully validate redirects.
- Segment networks so app servers cannot reach sensitive internal services unnecessarily.
