# CWE-295: Improper Certificate Validation

**Official URL:** https://cwe.mitre.org/data/definitions/295.html

## Description

The software does not validate, or incorrectly validates, a certificate. This can allow a man-in-the-middle to spoof a trusted entity by using a certificate that is not issued by a valid CA (including self-signed certificates when validation is disabled).

## Mitigations

- Enable full TLS certificate chain validation for outbound connections.
- Do not disable verification for convenience in production webhook delivery.
- Pin certificates only with a robust rotation process if used.
