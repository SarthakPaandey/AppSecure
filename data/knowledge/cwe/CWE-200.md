# CWE-200: Exposure of Sensitive Information to an Unauthorized Actor

**Official URL:** https://cwe.mitre.org/data/definitions/200.html

## Description

The product exposes sensitive information to an actor that is not explicitly authorized to have access to that information. GraphQL introspection in production can expose schema details useful to attackers.

## Mitigations

- Disable GraphQL introspection and schema suggestions in production.
- Minimize information in API responses and errors.
- Apply field-level authorization for sensitive GraphQL fields.
