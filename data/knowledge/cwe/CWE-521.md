# CWE-521: Weak Password Requirements

**Official URL:** https://cwe.mitre.org/data/definitions/521.html

## Description

The product does not require that users choose strong passwords, which makes it easier for attackers to compromise user accounts.

## Mitigations

- Enforce minimum length (e.g., 12+ characters) and modern password guidance.
- Check passwords against breached password lists.
- Support password managers (allow long passphrases and special characters).
- Prefer MFA rather than complex composition rules alone.
