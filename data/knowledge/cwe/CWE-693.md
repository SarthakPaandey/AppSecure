# CWE-693: Protection Mechanism Failure

**Official URL:** https://cwe.mitre.org/data/definitions/693.html

## Description

The product does not use or incorrectly uses a protection mechanism that provides sufficient defense against directed attacks against the product. Missing security headers is a common instance in web apps.

## Mitigations

- Add security headers: Content-Security-Policy, Strict-Transport-Security, X-Content-Type-Options, Referrer-Policy, Permissions-Policy.
- Configure headers via reverse proxy or application middleware consistently.
- Test header presence in CI and production monitors.
