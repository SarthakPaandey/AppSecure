# CWE-287: Improper Authentication

**Official URL:** https://cwe.mitre.org/data/definitions/287.html

## Description

When an actor claims to have a given identity, the software does not prove or insufficiently proves that the claim is correct. Includes accepting JWT tokens with algorithm 'none' or failing to verify signatures.

## Mitigations

- Explicitly allowlist accepted JWT algorithms; reject 'none'.
- Always verify cryptographic signatures with trusted keys.
- Use well-vetted authentication libraries; avoid custom crypto.
- Combine with MFA and strong session management where appropriate.
