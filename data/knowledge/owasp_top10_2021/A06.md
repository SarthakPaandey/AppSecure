# A06:2021 – Vulnerable and Outdated Components

**Official URL:** https://owasp.org/Top10/A06_2021-Vulnerable_and_Outdated_Components/

Components run with the same privileges as the application. If a vulnerable component is exploited, the attack can facilitate serious data loss or server takeover.

## Prevention

- Remove unused dependencies and features
- Continuously inventory client and server components and versions
- Monitor CVE sources; use SCA tools
- Prefer official sources over untrusted package mirrors
- Patch/upgrade in a risk-based, timely manner
